# JavaScript Style Guide
+ 參考資料：
    + MDN
    + Google JavaScript Style Guide
    + Airbnb JavaScript Style Guide
    + StandardJS
    + ESLint / Prettier 預設
+ 建議使用[格式化工具]
    + Prettier
    + ESLint
    + VS Code Auto Format
+ condition：
```js
// () 跟 {} 的前後都有空格
if () {
    ...
} else if {
    ...
}
```
+ function
```js
// 函式名稱後不留空格
// () 和 { 之間有一個空格
function new_function() {

}
```
+ arrow function
```js
// = 前後要空格
// => 前後都要空格
// { 前要空格
const fn = () => {
}
```
